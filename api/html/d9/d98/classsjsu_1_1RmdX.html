Detailed Description

Device driver class for the RMD-X series of smart servos. The RMD-X servos have precision speed and position control and many forms of feedback. The RMD-X smart servo motors handle all of the motor control in its own MCU, freeing up the host MCU to focus on application level tasks.

Find out more at this website: https://www.myactuator.com/
