float sjsu::RmdXSettings_t::gear_ratio = 8.0f

Reducer gear ratio. The RMD-X series has the encoder that reads rotation on the driving motor and not the output shaft of the motor. So as an example, if the gear ratio for the motor is 6:1, a request to rotate the motor by 360 rotation only accounts for 1/6 rotation of the output motor shaft. In order for the commanded rotation of the motor to equal the rotation output shaft this field must be set correctly.
